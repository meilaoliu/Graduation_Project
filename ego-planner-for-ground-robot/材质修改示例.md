# Gazebo材质修改指南

## 1. 可用的内置材质

Gazebo提供了多种内置材质，可以直接使用：

### 基础颜色材质
```xml
<material>
  <script>
    <uri>file://media/materials/scripts/gazebo.material</uri>
    <name>Gazebo/Red</name>        <!-- 红色 -->
  </script>
</material>
```

### 常用材质列表
- `Gazebo/Red` - 红色
- `Gazebo/Green` - 绿色  
- `Gazebo/Blue` - 蓝色
- `Gazebo/Yellow` - 黄色
- `Gazebo/Orange` - 橙色
- `Gazebo/Purple` - 紫色
- `Gazebo/White` - 白色
- `Gazebo/Black` - 黑色
- `Gazebo/Grey` - 灰色

### 材质纹理
- `Gazebo/Concrete` - 混凝土
- `Gazebo/Wood` - 木纹
- `Gazebo/Metal` - 金属
- `Gazebo/Brick` - 砖块
- `Gazebo/Stone` - 石材
- `Gazebo/Grass` - 草地
- `Gazebo/WoodFloor` - 木地板
- `Gazebo/CeilingTiled` - 瓷砖
- `Gazebo/PaintedWall` - 粉刷墙面

## 2. 修改围栏材质示例

### 将绿色围栏改为金属材质
```xml
<!-- 原来的围栏 -->
<material>
  <script>
    <uri>file://media/materials/scripts/gazebo.material</uri>
    <name>Gazebo/Green</name>
  </script>
</material>

<!-- 修改为金属材质 -->
<material>
  <script>
    <uri>file://media/materials/scripts/gazebo.material</uri>
    <name>Gazebo/Metal</name>
  </script>
</material>
```

### 使用自定义颜色
```xml
<material>
  <ambient>0.2 0.2 0.2 1</ambient>      <!-- 环境光反射 -->
  <diffuse>0.8 0.8 0.8 1</diffuse>      <!-- 漫反射 -->
  <specular>0.1 0.1 0.1 1</specular>    <!-- 镜面反射 -->
  <emissive>0 0 0 1</emissive>          <!-- 发光 -->
</material>
```

## 3. 修改地面材质

### 将混凝土地面改为草地
```xml
<!-- 在ground_plane模型中修改 -->
<material>
  <script>
    <uri>file://media/materials/scripts/gazebo.material</uri>
    <name>Gazebo/Grass</name>
  </script>
</material>
```

## 4. 为设备添加真实材质

### 变压器金属外壳
```xml
<material>
  <script>
    <uri>file://media/materials/scripts/gazebo.material</uri>
    <name>Gazebo/Metal</name>
  </script>
</material>
```

### 配电柜工业蓝色
```xml
<material>
  <ambient>0.1 0.2 0.5 1</ambient>
  <diffuse>0.2 0.4 0.8 1</diffuse>
  <specular>0.3 0.3 0.3 1</specular>
</material>
```

## 5. 使用自定义纹理

### 创建自定义材质脚本
```bash
# 在 ~/.gazebo/materials/scripts/ 创建 custom.material
material Custom/Fence
{
    technique
    {
        pass
        {
            ambient 0.5 0.5 0.5
            diffuse 0.8 0.8 0.8
            texture_unit
            {
                texture fence_texture.png
                scale 1.0 1.0
            }
        }
    }
}
```

### 在SDF中使用自定义材质
```xml
<material>
  <script>
    <uri>file://media/materials/scripts/custom.material</uri>
    <name>Custom/Fence</name>
  </script>
</material>
```
